# Users endpoinds

Эта папка содержит настройки и эксплуатацию моделей пользователя

## Структура
- `repositories/` - Начальный слой для получения, обработки, удаления данных
- `usecases/` - Вторичный слой для проверки моделей и их улучшенной обработке
- `routers/` - Третичьный слой для вывода информации
*Взаимодействие по средствам rt->us->repo->us->rt* # как мидлвар почти, только архитектурный
Такая структура необходима для точной проверки данных и разделения ответственности

## Endpoinds
# auth
`auth/register` - создание пользователя
`auth/token` - получение аксес через рефреш
`auth/check` - проверка наличия аксесс токена
`auth/refresh` - рефреш аксес токена через рефреш
`auth/ws-token` - получение токена для вебсокет соединения 
`auth/logout` - удаление токена
`auth/verified_email` - подтверждение почты
# user
`/users/` - получение всех пользователей
`/users/me` - получение своего пользователя 
`/users/me` - обновление своего пользователя
`/users/{user_id}` - получение пользователя по айди
